Usage

HTML (head section)

    <!--  if custom classes used -->
    <link rel="stylesheet" href="/path/to/custom.css">

    <script type="text/javascript" src="/path/to/jquery.js" type="text/javascript"></script>
    <script type="text/javascript" src="/path/to/jquery-ui.js" type="text/javascript"></script>

    <!--  if touch event support for touch devices is needed  -->
    <script src="/path/to/jquery.ui.touch-punch.min.js" type="text/javascript"></script>

    <script type="text/javascript" src="/path/to/jquery.PLUGIN.min.js" type="text/javascript"></script>

Methods list

COMMON

getDefaults

    $("#element_id").PLUGIN('getDefaults');

getOption

    $("#element_id").PLUGIN('getOption', 'option_name');

getAllOptions

    $("#element_id").PLUGIN('getAllOptions');

setOption

    $("#element_id").PLUGIN('setOption', 'option_name', option_value, reinit);

Alternative way to set one or more options 

    $("#element_id").PLUGIN({option1_name: option1_value, etc});

refresh

    $("#element_id").PLUGIN('refresh');

destroy

    $("#element_id").PLUGIN('destroy');
